what is application?
	-It is simple blogging app.
	-users can login/register
	-users can write blogs
	-users can read blogs
	-users can comment on blogs
	
links==>
	-login
	-register
	-logout
	-link for each blog
	-add comment
	-edit comment
	-delete comment
	-write new blog
	-edit blog
	-delete blog
	-my blogs
	
database tables ==>
	-users (default)
	-posts (id, author, title, body, slug, published_on, last_modified, active)
	-comments (id, on_post, from_user, body, at_time)
	
navbar content
	-logout
		login, register, home, my blogs, new blogs
	-login
		logout, home, my blogs, new blog

View ==>
	-auth (default)
	-posts
		-create
		-show
		-edit
	-user
		-profile
		-edit-profile
